Webhooks

Overview

When one of these events is fired, we'll send you the json in your webhook url that you have provided. 


    { 
        if($json = json_decode(file_get_contents("php://input"),true))
        {

        $response = $json;

        }
        else {

        $response = $_POST;
        
        }
    }

Events

In the Webhook panel you can select which events you would like to receive.
Event Description
call.new New call is generated.
call.scheduled Customer has scheduled a call.
call.missed There are no available agents.
call.claimed Call is claimed by an agent.
call.connected Call is connected to the customer.
call.finished Call is successfully completed.
